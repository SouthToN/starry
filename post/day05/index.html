一、旋转数组的最小数字

把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。输入一个递增排序的数组的一个旋转，输出旋转数组的最小元素。例如，数组 [3,4,5,1,2] 为 [1,2,3,4,5] 的一个旋转，该数组的最小值为1。

示例 1：
输入：[3,4,5,1,2]
输出：1

示例 2：
输入：[2,2,2,0,1]
输出：0

来源：力扣：旋转数组中的最小数字

思路：递增排序的数组，开始元素最小，末尾元素最大，旋转后最小元素则被搬到了最大元素之后,这是最直接的特点，由此入手。
code1：

class Solution {
    public int minArray(int[] numbers) {
        for(int i=0;i<numbers.length-1;i++)
        {
            if(numbers[i]>numbers[i+1])
                return numbers[i+1];
        }
        return numbers[0];
    }
}

二、矩阵中的路径😐

请设计一个函数，用来判断在一个矩阵中是否存在一条包含某字符串所有字符的路径。路径可以从矩阵中的任意一格开始，每一步可以在矩阵中向左、右、上、下移动一格。如果一条路径经过了矩阵的某一格，那么该路径不能再次进入该格子。例如，在下面的3×4的矩阵中包含一条字符串“bfce”的路径（路径中的字母用加粗标出）。
[["a","b","c","e"],
["s","f","c","s"],
["a","d","e","e"]]
但矩阵中不包含字符串“abfb”的路径，因为字符串的第一个字符b占据了矩阵中的第一行第二个格子之后，路径不能再次进入这个格子。

示例 1：
输入：board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "ABCCED"
输出：true

示例 2：
输入：board = [["a","b"],["c","d"]], word = "abcd"
输出：false

来源：力扣：矩阵中的路径

code1：

class Solution {
    boolean[][] visit;
    private int[][] dir= {{-1,0},{0,-1},{0,1},{1,0}};
    public boolean exist(char[][] board, String word) {
        int rows = board.length;
        int cols = board[0].length;
        visit = new boolean[rows][cols];
        for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
                if (word.charAt(0) == board[i][j] && dfs(board, word, i, j, 0)) {
					return true;
				}
			}
		}
		return false;
    }
    private boolean dfs(char[][] board, String word, int i, int j, int index) {
		if (index == word.length()) {
			return true;
		}
		if (i >= board.length || i < 0 || j >= board[0].length || j < 0 || visit[i][j] || word.charAt(index) != board[i][j]) return false;
		visit[i][j] = true;
		for (int[] d : dir) {
			if (dfs(board, word, i + d[0], j + d[1], index + 1))  return true;
		}
		visit[i][j] = false;
		return false;
	}
}
