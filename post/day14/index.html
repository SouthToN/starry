一、包含min函数的栈

定义栈的数据结构，请在该类型中实现一个能够得到栈的最小元素的 min 函数在该栈中，调用 min、push 及 pop 的时间复杂度都是 O(1)。

示例:
MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.min(); --> 返回 -3.
minStack.pop();
minStack.top(); --> 返回 0.
minStack.min(); --> 返回 -2.

来源：力扣：包含min函数的栈

思路：维护一个辅助栈
code1：

public class MinStack {
	private Stack<Integer> stack = null; // MinStack栈
	private Stack<Integer> min = null; // 辅助栈
	public MinStack() {
        stack=new Stack<>();
		min=new Stack<>();
    }
	// 压栈
	public void push(int x) {
		stack.push(x);
		if (min.isEmpty()) {
			min.push(x);
		} else {
			int temp = min.peek();
			// 如果x是最小值，则将x放在栈顶
			if (x <= temp) temp = x;
			min.push(temp);
		}
	}
	// 弹栈
	public void pop() {
		stack.pop();
		min.pop();//入栈时重复入过最小值故可以直接弹
	}
	// 取出栈顶元素
	public int top() {
		return stack.peek();
	}
	public int min() {
		return min.peek();
	}
}

二、栈的压入、弹出序列

输入两个整数序列，第一个序列表示栈的压入顺序，请判断第二个序列是否为该栈的弹出顺序。假设压入栈的所有数字均不相等。例如，序列 {1,2,3,4,5} 是某栈的压栈序列，序列 {4,5,3,2,1} 是该压栈序列对应的一个弹出序列，但 {4,3,5,1,2} 就不可能是该压栈序列的弹出序列。

示例 1：
输入：pushed = [1,2,3,4,5], popped = [4,5,3,2,1]
输出：true
解释：我们可以按以下顺序执行：
push(1), push(2), push(3), push(4), pop() -> 4,
push(5), pop() -> 5, pop() -> 3, pop() -> 2, pop() -> 1

示例 2：
输入：pushed = [1,2,3,4,5], popped = [4,3,5,1,2]
输出：false
解释：1 不能在 2 之前弹出。

来源：力扣：栈的压入、弹出序列

思路：借助一个辅助栈，将入栈序列逐个压入辅助栈，判断辅助栈的栈顶元素是否等于出栈序列的弹出数字，是则直接弹出，若不是，则把压栈序列中还没有入栈的数字压入辅助栈。辅助栈每弹栈一次，index自增一，入栈序列全部压入辅助栈后，index等于出栈序列元素个数，则返回true。
code1：

class Solution {
    public boolean validateStackSequences(int[] pushed, int[] popped) {
        Stack<Integer> stack = new Stack<>();
		int index = 0;
		for (int i = 0; i < popped.length; i++) {
			stack.push(pushed[i]);
            //判断辅助栈的栈顶元素等于弹出序列的第index元素
			while (!stack.isEmpty() && popped[index] == stack.peek()) {
				stack.pop();
				index++;
			}
		}
		return index == popped.length;
    }
}
