一、n个骰子的点数

把n个骰子扔在地上，所有骰子朝上一面的点数之和为s。输入n，打印出s的所有可能的值出现的概率。
你需要用一个浮点数数组返回答案，其中第 i 个元素代表这 n 个骰子所能掷出的点数集合中第 i 小的那个的概率。

示例 1:
输入: 1
输出: [0.16667,0.16667,0.16667,0.16667,0.16667,0.16667]

示例 2:
输入: 2
输出: [0.02778,0.05556,0.08333,0.11111,0.13889,0.16667,0.13889,0.11111,0.08333,0.05556,0.02778]
限制：1 <= n <= 11

来源：力扣：n个骰子的点数

思路：动态规划
F(n,s)=F(n-1,s-1)+F(n-1,s-2)+F(n-1,s-3)+F(n-1,s-4)+F(n-1,s-5)+F(n-1,s-6)
code:

class Solution {
    public double[] twoSum(int n) {
        if (n < 1)   return new double[0];
        double[] dp = new double[6*n + 1];
        double[] pr = new double[5*n + 1];//可能的点数和
        double sum=Math.pow(6, n);        
        for (int i = 1; i <= 6; i++) {
            dp[i]=1;
            pr[i-1]=1.0/6;            
        }
        for (int i = 2; i <= n; i++) {  //控制骰子个数
            for (int j = n*6; j >=1; j--) {
                int temp=0;
                for (int k = 1; k <= 6; k++) {
                    temp+=j>=k?dp[j-k]:0;
                }
                dp[j]=temp;
                if (i==n&&j>=n) pr[j-i]=dp[j]/sum;                
            }
        }
        return pr;
    }
}

二、 扑克牌中的顺子

从扑克牌中随机抽5张牌，判断是不是一个顺子，即这5张牌是不是连续的。2～10为数字本身，A为1，J为11，Q为12，K为13，而大、小王为 0 ，可以看成任意数字。A 不能视为 14。

示例 1:
输入: [1,2,3,4,5]
输出: True

示例 2:
输入: [0,0,1,2,5]
输出: True

限制：数组长度为 5 & 数组的数取值为 [0, 13] .

来源：力扣：扑克牌中的顺子

思路：排序，统计数组中0的个数，数组中若有两个数相等则不能构成顺子，数组中不连续的两个数之间缺的数字个数必须小于等于数组中0的个数
code：

class Solution {
    public boolean isStraight(int[] nums) {
        if(nums.length<5) return false;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] < 0 || nums[i] > 13) return false;               
        }
        int zero = 0;
        Arrays.sort(nums);
        for (int i = 0; i < nums.length - 1; i++) {
            if (nums[i] == 0) {
                zero++;
                continue;
            }
            if (nums[i] == nums[i + 1]) return false;                
            if (nums[i] != nums[i + 1] + 1) zero -= nums[i + 1] - nums[i] - 1;                
        }
        return zero >= 0;
    }
}
