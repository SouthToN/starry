一、求1+2+…+n

求 1+2+...+n ，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。

示例 1：
输入: n = 3
输出: 6

示例 2：
输入: n = 9
输出: 45

限制：

  • 1 <= n <= 10000

来源：力扣：求1+2+…+n

思路：使用递归，条件与 && 在第一个条件语句为 false 的情况下不会去执行第二个条件语句。
code：

class Solution {
    public int sumNums(int n) {
        // return n*(n+1)/2;
        int sum = n;
        boolean b = (n > 0) && ((sum += sumNums(n - 1)) > 0);
        return sum;
    }
}

二、不用加减乘除做加法

写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“*”、“/” 四则运算符号。

示例:
输入: a = 1, b = 1
输出: 2

提示：

  • a, b 均可能是负数或 0
  • 结果不会溢出 32 位整数

来源：力扣：不用加减乘除做加法

思路：位运算
code：

class Solution {
    public int add(int a, int b) {
        while(b != 0){
            int temp = a ^ b; //按位异或
            b = (a & b) << 1; //进位
            a = temp;
        }
        return a;
    }
}
