一、调整数组顺序使奇数位于偶数前面

输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有奇数位于数组的前半部分，所有偶数位于数组的后半部分。

示例：
输入：nums = [1,2,3,4]
输出：[1,3,2,4]
注：[3,1,2,4] 也是正确的答案之一。

来源：力扣：调整数组顺序使奇数位于偶数前面

思路1：直接双循环遍历
code1:

class Solution {
    public int[] exchange(int[] nums) { // 时间复杂度太高
		for (int i = 0; i < nums.length; i++) {
			for (int j = 0; j < nums.length; j++) {
				if (nums[i] % 2 == 1 && nums[j] % 2 == 0) {
					int temp = nums[j];
					nums[j] = nums[i];
					nums[i] = temp;
				}
			}
		}
		return nums;
	}
}

思路二：空间换时间
code2:

class Solution {
    public int[] exchange(int[] nums) { // 空间换时间
		int[] exnums = new int[nums.length];
		int countOdd = 0;// 统计奇数个数
		for (int i = 0; i < nums.length; i++) {
			if (nums[i] % 2 != 0)
				countOdd++;
		}
		if (countOdd == 0)
			return nums;
		int a = 0;
		for (int i = 0; i < nums.length; i++) {
			if (nums[i] % 2 == 0) {
				exnums[countOdd++] = nums[i];
			} else {
				exnums[a++] = nums[i];
			}
		}
		return exnums;
	}
}

思路三：双指针(高效)
code3:

class Solution {
    public int[] exchange(int[] nums) {       
		int start = 0, end = nums.length - 1;
		while (start < end) {			
			while (start < nums.length && nums[start] % 2 == 1) {// 从前往后找偶数
				start++;
			}
			while (end >= 0 && nums[end] % 2 == 0) {// 从后往前找奇数
				end--;
			}
			if (start < end) {//奇偶对调
				int temp = nums[start];
				nums[start] = nums[end];
				nums[end] = temp;
			}

		}
		return nums;
    }
}

二、链表中倒数第k个节点

输入一个链表，输出该链表中倒数第k个节点。为了符合大多数人的习惯，本题从1开始计数，即链表的尾节点是倒数第1个节点。例如，一个链表有6个节点，从头节点开始，它们的值依次是1、2、3、4、5、6。这个链表的倒数第3个节点是值为4的节点。

示例：
给定一个链表: 1->2->3->4->5, 和 k = 2.
返回链表 4->5.

来源：力扣：链表中倒数第k个节点

思路：直接遍历
code1:

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode getKthFromEnd(ListNode head, int k) {
        if (k < 0) {
			return null;
		}
		ListNode ln1 = head;
		ListNode ln2 = head;
		for (int i = 0; i < k && ln1 != null; ++i) {
			ln1 = ln1.next;
		}
		while (ln1 != null) {
			ln1 = ln1.next;
			ln2 = ln2.next;
		}
		return ln2;
    }
}
