2020-03-18 3 min read

一、判定字符是否唯一

实现一个算法，确定一个字符串 s 的所有字符是否全都不同。
示例 1：
输入: s = "leetcode"
输出: false

示例 2：
输入: s = "abc"
输出: true
限制：

  • 0 <= len(s) <= 100
  • 如果你不使用额外的数据结构，会很加分。
    来源：力扣：判定字符是否唯一

思路1：最直接的方法，暴力遍历
code1：

class Solution {
    public boolean isUnique(String astr) {
        if(astr.length()==0) return true;
        for (int i = 0; i < astr.length(); i++) {            
            for (int j = i+1; j < astr.length(); j++) {
                if(astr.charAt(i)==astr.charAt(j))return false;            }
        }
        return true;
    }
}

思路2：利用hashset，如果字符唯一则最后hashset的长等于字符串的长度。
code2：

class Solution {
    public boolean isUnique(String astr) {
        if(astr.length()==0) return true;
        HashSet uni = new HashSet<String>();
        for (int i = 0; i < astr.length(); i++) {
            uni.add(astr.charAt(i));
        }
        return uni.size()==astr.length();
    }
}

二、判定是否互为字符重排

给定两个字符串 s1 和 s2，请编写一个程序，确定其中一个字符串的字符重新排列后，能否变成另一个字符串。
示例 1：
输入: s1 = "abc", s2 = "bca"
输出: true

示例 2：
输入: s1 = "abc", s2 = "bad"
输出: false
说明：

  • 0 <= len(s1) <= 100
  • 0 <= len(s2) <= 100

来源：力扣：判定是否互为字符重排

思路1：将字符串转为字符串数组并排序，然后逐个比较排序后字符是否相等
code1：

class Solution {
    public boolean CheckPermutation(String s1, String s2) {
        if (s1.length() != s2.length()) return false;
        char[] c1 = s1.toCharArray();
        char[] c2 = s2.toCharArray();
        Arrays.sort(c1);
        Arrays.sort(c2);
        for (int i = 0; i < c1.length; i++) {
            if(c1[i]!=c2[i]) return false;
        }
        return true;
    }
}

思路2：将字符串中的字符转换为ASCII码并求和，判断两个字符串的ASCII码和是否相等
code2：

class Solution {
    public boolean CheckPermutation(String s1, String s2) {
        if (s1.length() != s2.length()) return false;
        int sum1=0,sum2=0;       
        for (int i = 0; i < s1.length(); i++) {            
            sum1+=s1.charAt(i)-'a';
            sum2+=s2.charAt(i)-'a';
        }
        return sum1==sum2;
    }
}
