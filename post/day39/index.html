一、判断能否形成等差数列

给你一个数字数组 arr 。
如果一个数列中，任意相邻两项的差总等于同一个常数，那么这个数列就称为 等差数列 。
如果可以重新排列数组形成等差数列，请返回 true ；否则，返回 false 。
示例 1：
输入：arr = [3,5,1]
输出：true
解释：对数组重新排序得到 [1,3,5] 或者 [5,3,1] ，任意相邻两项的差分别为 2 或 -2 ， 可以形成等差数列。

示例 2：
输入：arr = [1,2,4]
输出：false
解释：无法通过重新排序得到等差数列。

来源：力扣：判断能否形成等差数列

思路：力扣196场周赛的第一题，看到这题第一想法就是排序+计算数列的公差，这里我用了快排对数组进行排序，使用HashSet来存储数组的公差，如果是等差数组则set中只会有一个值，即就是数字的公差。
code：

class Solution {
   public static boolean canMakeArithmeticProgression(int[] arr) {
        int l=0;
        int r=arr.length-1;
        quickp(arr,l,r);        
        HashSet<Integer> cha = new HashSet<Integer>();
        for(int i = 0;i<r;i++){
           int temp =arr[i+1]-arr[i];
           cha.add(temp);
        }
        if(cha.size()==1){
            return true;
        } else{
            return false;
        }
       
    }
    
    static void quickp(int[] nums,int l, int r){        
        int left = l, right = r;        
        while(left<right){
            int mid = nums[left];
            while(left<right && nums[right]>=mid)  right--;  //找到右边比mid值小的位置            
            nums[left] = nums[right];            
            while(left<right && nums[left]<mid) left++; //找到左边比mid值大的位置  
            nums[right] = nums[left];
            nums[left] = mid;
            quickp(nums, l, left);
            quickp(nums, left+1, r);
        }
    }
}
